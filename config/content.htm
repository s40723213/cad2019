Week 2
建立新的github倉儲

1.到個人網站建立新的倉儲，並且clone到近端(要去mde.tw下載"2019fall可攜套件")。

2.用git config --global user設定名稱、信箱和proxy，之後再用指令git submodule add 建立目錄取下子模型。

3.最後再輸入python -m pip install flask cors即可。

4.備註:要進入近端，必須先進入cmsimde，指令python wsgi.py方能使用。

Week3
Solvespace編譯

1.Y:\portablegit\bin\sh.exe 改名為 sh_rename_for_solvespace.exe。

2.更新github版本到2.13以上。

3.git clone --recurse-submodules https://github.com/solvespace/solvespace.git solvespace

4.編輯cmakelist，在713~714行之間加入#字號。

5.把ibpng.dll.a 的編譯, 並改名為 libpng_static.a, 並放到編譯系統的 lib 目錄中。

6.切路徑至build，再執行cmake .. -G "MinGW Makefiles" -DCMAKE_BUILD_TYPE=Release，輸入mingw32-make。

7.路徑切回solvespace原始碼，建立並進入Build目錄，再次執行cmake .. -G "MinGW Makefiles" -DCMAKE_BUILD_TYPE=Release，輸入mingw32-make。
